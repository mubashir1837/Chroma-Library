A modern Visual Studio Code theme pack created for developers who prefer vivid aesthetics and clean coding environments. Includes support for custom bioinformatics-themed UI and backgrounds.

---

## Features

- Multiple unique high-contrast and soft-light themes
- DNA Helix mode tailored for bioinformatics enthusiasts
- Fully customizable color palette for syntax and UI
- Support for animated backgrounds (DNA helix motif)
- Designed with readability and focus in mind

---

## Installation

You can install the extension directly from the Visual Studio Code Marketplace:

### From VS Code

1. Open Extensions (`Ctrl+Shift+X`)
2. Search for `Chroma Library`
3. Click **Install**

### Via Command Line

```bash
code --install-extension codewithbismillah.chroma-library
